Web Amigo

Uma extensão simples criada pelo aluno Enzo cortes

Funcionalidades

  1. Extensão criada para interação humana com a maquina

    2. Como Baixar

  2. Clone o repositório no seu computador.
  3. Abra chrome://extensions no Chrome/Edge (ou navegador compatível).
  4. Ative o Modo de desenvolvedor e clique em Carregar sem compactação.
  5. Selecione a pasta do projeto (a pasta raiz que contém manifest.json).

O popup estará disponível no ícone da extensão na barra do navegador.
